Протягом тижня виконувались роботи по знаходженню інформації щодо можливого використання Instagram API.

Були зроблені такі кроки:

1. Ознайомлення з офіційною документацією Instagram API.
2. Налагодження доступу до API.
3. Розробка модуля для тестування.
4. Ознайомлення з модулями-обгортками над офіційним Instagram API.
5. Вибір найбільш вдалого.
6. Розробка модуля для тестування модуля-обгортки.
7. Прийнято рішення комбінувати можливості обгортки на офійної версії API.
