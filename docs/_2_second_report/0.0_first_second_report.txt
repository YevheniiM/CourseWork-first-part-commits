Протягом тижня виконувались роботи по знаходженню інформації щодо можливого використання Facebook API.

Були зроблені такі кроки:

1. Ознайомлення з офіційною документацією Facebook API.
2. Налагодження доступу до API.
3. Розробка модуля для тестування.
4. Ознайомлення з модулями-обгортками над офіційним Facebook API.
5. Розчарування з приводу можливосвтей оновленої версії API, а саме
   неможливість доступу додатку до друзів користувача навіть у випадку
   авторизації.
6. Знайшлося рішення для останьої проблеми: додаток може мати доступ до
   користувачів, які на цей момент авторизувалися в додатку.
   (Тобто, можливість додатку буде дещо обмежена, проте він продовжить
   повноцінно працювати).
